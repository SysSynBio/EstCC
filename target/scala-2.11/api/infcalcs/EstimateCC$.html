    Calculates channel capacity (maximum MI) given list of MI estimates.

    Takes a list of MI estimates for varying weights, eg., as returned by calcWithWeightsMult. For every entry in the list, calls EstimateMI.optMIMult to find the unbiased MI, and then returns the maximum unbiased MI found in the list.
